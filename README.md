Overview
# Premium Haircare Retail Pricing & Promotions Analysis
This project explores how pricing, promotions and category mix impact performance across a premium haircare portfolio (example).
using SQL, I analyse sales and promotion data across categories including shampoo, conditioner, serums, oils, leave-in treatments, sprays and hair masks.
## Commcercial Questions explored
- which premium haircare categories generate the most revenue?
- how do promotions affect conversion and units sold?
- which promotion types perform best across categories?
- where are pricing and promotional opportunities being missed?
